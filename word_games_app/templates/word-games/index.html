WordGames

Welcome to WordGames! Test your English vocabulary skills with 3 challenging games.
Below you will find explanations to each game, log in or sign up to start playing.
Check the high scores to find out who was able to get the best results for each game, try to improve and reach the top.

In this game you have 6 guesses to try and find a 5-letter word.
Each guess must be a valid English word. If you guess a letter in its correct position, it will turn green.
If you guess a letter in the word but in the incorrect position, it will turn yellow. When you manage to guess the word, you earn points.
The fewer guesses it took you, the more points you earn!

For this game the goal is to find as many words as you can within one minute.
When you click the start button, a collection of random letters will appear on the board and the timer will start running. Submit as many words as you can find into the field below, using only the letters you see on screen. After one minute has passed, your points will be counted up.
To earn a point, make sure you only use the letters you see on screen. Only valid English words with at least 4 letters will be counted.

In this game you try to make as many unique 5-letter words as you can.
Start by typing your first word in the input field, then press start to begin the game. When you submit a word using the start button, one letter will be frozen in place.
Now it's up to you to make the next word using this frozen letter. Only valid 5-letter English words are accepted.
You have 90 seconds to form as many unique words as you can. Each word earns you 1 point.
